# Radio

## Basic Usage with `useRadioState` Hook

> Previously in PCR 1.0 the `name` prop was required. This restirction has been removed in 2.0

```jsx live
function App() {
    const radio = useRadioState();

    return (
        <>
            <Radio {...radio} value="apple">
                Apple
            </Radio>
            <Radio {...radio} value="orange">
                Orange
            </Radio>
            <Radio {...radio} value="watermelon">
                Watermelon
            </Radio>
        </>
    );
}
```

## Disabled and Locked Radio

```jsx live
<>
    <Radio disabled>Disabled</Radio>
    <Radio locked>Locked</Radio>
</>
```

## RadioGroup

PCR 2.0 introduces a new component called `RadioGroup`.
It's just a wrapper around HTML `fieldset` that pre-configures ARIA `role` for a radio group.

```jsx live
function App() {
    const radio = useRadioState();

    return (
        <RadioGroup arial-label="fruits">
            <Radio {...radio} value="apple">
                Apple
            </Radio>
            <Radio {...radio} value="orange">
                Orange
            </Radio>
            <Radio {...radio} value="watermelon">
                Watermelon
            </Radio>
        </RadioGroup>
    );
}
```
